## 외부 링크 (External) — 1개

| API | ID | 설명 | 링크 |
|-----|-----|------|------|
| 국토교통부 국토지리정보원_영상지도, 배경지도 API | [15059358](https://www.data.go.kr/data/15059358/openapi.do) | Open-API 서비스를 국토지리정보원이 보유한 최신 공간정보를 기반으로 배경 지도와 국가관심지점정보(POI)를 구축하여 다른 정보시스템에서 연계·활용할 수 있습니다.  Open-API 서비스는 배경지도(일반지도, 다국어, 영상지도, 위성지도, 색양자지도, 큰글씨, 백지도, 교육용 백지도, 야간지도, 하이브리드지도를 포함)와 검색API(POI, 지명, 기준점, 지오코더) 두 가지 형태로 제공됩니다. 웹사이트 개발 및 운영 시 별도의 지도정보 구축 없이 DHTML 및 Javascript 등을 활용해 공간정보 및 위치 검색 기능을 구현할 수 있습니다.  Open-API 서비스는  WMTS(Web Map Tile Service) 방식으로 제공되며, 국토정보플랫폼 오픈API 서비스는 회원을 위한 서비스입니다. 로그인(회원가입) 후 진행해 주시기 바랍니다.  (신청과정) 인증 사용자 및 기관등록 → 인증키 발급 신청  →  인증키 발급 → 서비스 실시 → API 사용자 내역관리 | [링크](<http://map.ngii.go.kr/mi/openKey/openKeyRegist.do>) |
